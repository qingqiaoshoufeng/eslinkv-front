## 快捷键
![](/node/docs-static/keyboard-1.png)

### 清除全部参考线
`Alt` + `C`

### 放大画布
`Ctrl` + `+`

### 缩小画布
`Ctrl` + `-`

### 拖动画布
`空格` + `左键`

### 水平移动画布
`Shift` + `滚轮`

### 垂直移动画布
`滚轮`
